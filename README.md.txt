Titanic Survival Prediction

Overview

This project implements a machine learning classification model to predict whether a passenger survived the Titanic disaster. The model is trained using the Titanic dataset and applies data preprocessing, feature selection, and classification techniques to generate predictions.

Objectives

* Load and analyze the Titanic dataset.
* Perform data preprocessing and handle missing values.
* Train a Decision Tree Classification model.
* Evaluate model performance using accuracy metrics.
* Visualize results using graphical representations.

Technologies Used

* Python
* Pandas
* Matplotlib
* Scikit-learn

 Dataset

The project uses the Titanic dataset from Kaggle, containing passenger information such as age, gender, passenger class, and fare.

Machine Learning Algorithm

Decision Tree Classifier

## Results

The trained model predicts passenger survival and displays the classification accuracy through graphical visualization.

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the project:
   python main.py
